Bonnes pratiques git:

(1) - Ajout d'un fichier sur ton commit et définition de ton message de commit
1 - git add ton_fichier.py
2 - git commit -m "Ajout tonfichier.py"

(2) envoi du fichier sur ton répertoire distant
3 - git push

(3) voir l'état de tes commits en attente
git status

(4) récupérer la dernière version de ton répertoire distant
git pull
